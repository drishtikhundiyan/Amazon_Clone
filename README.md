# Amazon Clone

## 📌 Description
A responsive Amazon-inspired web application built using HTML, CSS, and JavaScript. It replicates core UI components such as the navigation bar, product sections, and homepage layout with basic interactivity.

## 🚀 Features
- Responsive navigation bar with search and links  
- Product listing sections (Featured Products, Deals)  
- Dynamic product rendering using JavaScript  
- Clickable buttons and basic user interactions  
- Clean and structured homepage layout  

## 🛠️ Tech Stack
- HTML  
- CSS  
- JavaScript  

## ▶️ How to Run
1. Clone or download the repository  
2. Open `index.html` in your browser
